# iyanls
My custom build of `ls` in Rust.

## Features
- Colorful output
- File type (directory / file)
- File Permissions
- File Size
- Modification Time
- Grabbing files by strings

## Usage
```
iyanls [path] [options]
```
Options:
- `-h` `--help`: Print help page
- `-V` `--version`: Print version
- `-g` `--grab`: Grab files that match a string
